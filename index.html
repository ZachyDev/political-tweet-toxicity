Political Toxicity

Visualizations of toxicity online towards politicians

Rina Schiller, Stephanie Polkinghorne, and Tom Shanahan

About the Project

This project displays data visualizations about toxicity on Twitter for various high profile politicians. The data was collected by querying the Twitter API periodically and dates back to early 2019. We queried the Twitter API and searched for key words including the politician's first and last name and their twitter handles. In order to get information about the sentiment of the tweet, we used the Perspective API, an API used to detect abuse and harassment online. Perspective API has the ability to detect multiple attributes. We used TOXICITY, IDENTITY ATTACK, and SEXUALLY EXPLICIT.

Gender and Targeting of Explicit Tweets

Women in public life face a variety of challenges that their male counterparts do not. Notable among these are sexually explicit insults and attacks. This is exemplified in Twitter data of sexually explicit tweets sent to female politicians. While the data is live and subject to current events and reactions, tweets to female politicians consistently appear higher in sexually explicit attributes. This especially the case at the very high end of the range with very explicit tweets.

Female politicians include: Amy Klobuchar, Alexandria Ocasio-Cortez, Nancy Pelosi, Elizabeth Warren, Kamala Harris, Susan Collins, Lisa Murkowski

Male politicians include: Joe Biden, Pete Buttigieg, Chuck Schumer, Bernie Sanders
